
# QBERT

QBERT is a hybrid quantum-classical system for software defect prediction. It combines quantum machine learning concepts with NLP embeddings extracted using CodeBERT to analyze and classify source code.

## Project Overview

QBERT processes CLS vector embeddings extracted from the Defectors dataset using CodeBERT and performs binary classification:

- Defective code  
- Non-defective code  

The system explores multiple quantum machine learning configurations including:
- Encoding strategies  
- Variational ansätze  
- Measurement techniques  


## Web Deployment

This project is deployed as a web application that allows:
- Input or upload of code samples  
- Real-time prediction  
- Interactive visualization of results  

The web platform is used as a live demonstration of the QBERT system.


## Key Features

- Hybrid quantum-classical architecture  
- CodeBERT-based embeddings  
- Binary defect classification  
- Multiple quantum circuit configurations  
- Interactive web-based interface
